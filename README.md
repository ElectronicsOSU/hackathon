Electronics Club Hackathon Tools
================================

## How to use ##

This section describes using each individual tool designed for hackathons.
Many of these tools can be used for other purposes and many are easily
modifiable for use in other applications.

### Purchase Requests ###

Located in `./purchaseRequests`, this tool is designed to be run as an app in
Google Drive. It takes input from the user about a purchase request they have
and it uploads a document to Google Drive at a configurable location.
